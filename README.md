# WODWarriors

## 1. 목표와 기능

### 1.1 목표
#### 와드워리어즈 웹 애플리케이션은 현대 사회에서 사람들을 모아 함께 운동할 수 있도록, 운동 종류와 위치를 공유하는 플랫폼입니다.
- 
### 1 기능
- 로그인 & 회원가입
- 사용자가 원하는 운동 모임 개설 및 삭제
- 사용자가 지정한 위치를 기반으로 카카오맵에 운동 종류와 마커 표시
- 운동 모임 참여 및 미참여 선택 가능
